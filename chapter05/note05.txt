                                第5章 选择语句

C语言语句：
    ·选择语句
    ·重复语句
    ·跳转语句：break、continue、goto、return
    ·复合语句
    ·空语句

关系运算符
    产生的结果为0或1
    关系运算符允许比较混合类型的操作数
    优先级低于算术运算符
    左结合

判断运算符
    优先级低于关系运算符
    左结合
    技巧：依据i是否小于、大于或等于j，得出表达式(i>=j)+(i==j)的值分别是0、1和2。

逻辑运算符
    操作数为0或非0（任何非0的值作为真）
    产生的结果为0或1
    特性：短路计算
        如果表达式的值可以仅由左操作数推到出来，将不计算右操作数。
        副作用：
            eg: i > 0 && ++j > 0
            如果i > 0 表达式为假，将不会计算表达式 ++j > 0,
            那么j也不会进行自增。

级联式if语句
    不是新的语句类型，只是一种编写格式。
    悬空else语句
        else子句属于离它最近的且还未匹配的if语句。

条件表达式
    最好避免使用
        少数地方使用：return语句、调用printf函数......
    int i;float f;
    表达式(i > 0 ? i : f)的值为float型

C89布尔值
    没有定义布尔类型
    可以定义int型变量赋值0、1：int flag = 0;
    但通常使用TRUE和FALSE的名字定义宏：#define TRUE 1
    甚至可以定义一个可用做类型的宏：#define BOOL int

C99中的布尔值
    提供了_Bool类型：_Bool flag;（C89中，以下划线跟一个大写字母的名字为保留字，不应该使用）
    _Bool是无符号整型，但只能存储0或1，其他非0值也赋值为1，
    还提供了头文件：<stdbool.h>，该头文件提供了bool宏（注意是宏，不是类型），用来代表_Bool类型。
    <stdbool.h>头非常方便，需要使用布尔变量都用这个头。

switch语句
    ·控制表达式：不能用浮点数或字符串
    ·分支标号：不允许重复，但没有顺序（default不一定放最后），最后一个分支的break语句可省，但最好加上，
        常量表达式：不能包含变量或函数调用
